# Трекер бега

https://play.google.com/store/apps/details?id=org.producer.egor.run_tracker&hl=en_IN

[![N|Solid](https://github.com/EgorSidorov/Run_tracker/blob/master/release/new_run.jpg)]

### Описание возможностей приложения
Приложение написано с использованием библиотеки Qt для телефонов на ОС Android. Трекер позволяет сохранять историю бега по GPS сигналу, отображать трек разными цветами в зависимости от скорости(скорость задается в настройках), ставить время отложенного старта, считать расстояние, максимальную скорость и потерянные калории при беге. Приложение находится в стадии активной разработки.

[![N|Solid](https://github.com/EgorSidorov/Run_tracker/blob/master/release/history_run.jpg)]

### Особенности использования Qt
Данные о истории тренировок хранятся в СУБД SQLite.Приложение написано с использованием QML/JS и С++ функций для работы с БД и получения данных о качестве сигнала(качество сигнала определяется как сумма отношений сигнал/шум со спутников в децибелах с минимальным порогом). Для работы со специфичными для Android параметрами(уведомления и режим энергосбережения) используется Java Android API, вызываемая через интерфейс JNI.
